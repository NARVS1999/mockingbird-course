Lesson 0033: Character Analysis

Malalim na pagsusuri sa mga karakter — motivations, conflicts, at transformations.

Lesson Goal Matapos ang lesson na ito, mai-deconstruct mo ang motivations, conflicts, at transformations ng mga pangunahing karakter, at mai-classify sila bilang static/dynamic, round/flat.

Character Dossiers Activity

Bawat grupo ay gagawa ng "FBI File" para sa isang character. Pumili ng isa sa mga sumusunod:

Character Key Traits Static or Dynamic?
Scout Finch Curious, matapang, honest Dynamic — lumalim ang understanding niya sa mundo
Jem Finch Protective, idealistic, maturing Dynamic — nagbago ang paniniwala niya sa justice
Atticus Finch Moral, calm, principled Static — consistent ang kanyang values
Boo Radley Shy, kind, mysterious Dynamic — nagbago mula sa "halimaw" patungong tagapagligtas
Tom Robinson Kind, honest, empathetic Static — consistent ang kanyang kabutihan
Bob Ewell Lazy, abusive, racist Static — hindi nagbago
Mayella Ewell Lonely, sad, abused Dynamic — lumabas ang vulnerability niya
Calpurnia Strict, bridge between worlds Static — consistent ang role niya
Miss Maudie Kind, truthful, progressive Static — consistent ang values niya
Aunt Alexandra Proud, traditional, evolving Dynamic — nagbago ang tingin niya sa mga Finch

Character Dossier Project

Pumili ng isang character at gumawa ng "FBI File" gamit ang template sa itaas. I-post sa dingding at magbigay ng positive sticky note feedback sa iba.

Summary

  • Ang mga character ay may iba't ibang types: static/dynamic, round/flat
  • Si Scout at Jem ay dynamic — nagbago sila sa buong kwento
  • Si Atticus ay static — consistent ang kanyang values
  • Ang character analysis ay tumutulong sa pag-unawa sa theme at plot
Questions? Magtanong sa teacher tungkol sa character analysis o sa mga terms.
